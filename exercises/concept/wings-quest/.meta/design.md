## Goal

The goal of this exercise is to teach the student about Bools in Swift.

## Learning objectives

- Know what a Bool is (True and False)
- Know how to use `!`, `&&`, and `||`
- Know that you can affect the result of Bool with parentheses.

## Out of scope

- Using comparison operators
- If statements

## Concepts

`Bools`:

- Bools (`true`, `false`)
- `&&`, `||`, `!`
